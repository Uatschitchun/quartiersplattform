### Plugins

`Custom Post Type UI`

## Set up
* Setup local Wordpress 
* Clone Repository
`git clone https://github.com/studio-arrenberg/quartiersplattform.git` in `wp-content/themes`
* Sync Database with `WP Sync DB`
* Copy `wp-content` manually
